# Mythos Director Desktop App

Electron wrapper that launches the full Iron Jarvis + Mythos OS in a native window.

## Features

- Bundled daemon + Next.js dashboard
- Auto-updates from GitHub Releases
- System tray (always-on)
- Native file picker for LTM sources
- Voice input via OS speech APIs

## Build

pnpm --dir desktop run dist:full

## Production

The desktop app is the recommended way to run Mythos Director.
