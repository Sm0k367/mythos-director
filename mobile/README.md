# Mobile PWA + Native Apps

Mythos Director runs beautifully on mobile.

## PWA

- Installable from Chrome/Safari
- Offline-first with service worker
- Push notifications for reviews and schedules
- Same codebase as desktop (Next.js)

## Native (future)

- React Native or Tauri mobile builds
- Biometric auth for vault access
- Camera input for visual agents

## Status

PWA is production-ready. Native apps planned.
