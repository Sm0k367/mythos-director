# Workflow Engine (n8n-style)

Agents can create, edit, and run visual workflows.

## Features

- Drag-and-drop canvas in dashboard
- Nodes: Trigger, Gather, Mythos Core, Omnimedia Generate, Review, Deploy
- Agents can author workflows at runtime
- Persisted + versioned
- Full execution history
